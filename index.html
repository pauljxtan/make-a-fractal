Make a fractal

Fully client-side fractal visualization

Highlight a region to zoom in. Double-click to zoom back out.

The fractals here are drawn using a neat technique known as an iterated function system (IFS). Basically: for a given fractal, we have a set of transformations, each of which may be applied with some probability (e.g., 1 out of 10 times). These transformations are applied iteratively over and over - the more iterations, the more self-similarity emerges (and the cooler it looks).


The charting is done with the excellent dygraphs library.


The following fractals are currently implemented:

Fractal Recommended min. iterations (to make it look cool)
Barnsley's fern 20,000
Koch snowflake 50,000
